# Installation

## Requirements

- Python 3.9 or greater

## Install from pip

1. (recommended) Create a virtual environment (e.g. Mamba)
2. Install using pip:

```bash
pip install nemi-learn
```

## Install from source

1. Clone the repository:

```bash
git clone https://github.com/maikejulie/NEMI.git
```

2. (recommended) Create a virtual environment (e.g. Mamba)
3. Install using pip:

```bash
pip install -e .
```
